# ADR-004 · P0 贡献毛利率仅限 IHC/组织学线 —— 非 IHC 技术线一律出率

> **日期**：2026-07-04 | **状态**：已采纳（PM 拷问 Q6）| **性质**：hard-to-reverse（决定 P0 贡献毛利率覆盖哪些业务线）。
> **关联**：P0 spec §4 不变量⑧ / §10.C；ADR-002（Q4 同源锁）；`classifier.ts`；`lis_cases.business_line`（预埋未填）。

## 背景（PM Q6 已核）

- P0 收入分子 `lab_revenue = Σ(IN 结算)` 涵盖**所有 IN 业务线**（组织学 + 细胞病理 + 宫颈液基 + 冰冻 + 会诊…）。
- P0 成本减项白名单（§3）= 一抗/二抗显色/特染/组织处理料 —— **全是 IHC/组织学线材料**。
- **错配**：纯细胞病理/冰冻 case 有技术实收（`lab_revenue>0`）、但无抗体 marker（不做 IHC）→ 桶A=桶B≈0 → P0 减≈0 成本 → **毛利≈100%**。上卷到院级 → 含大量细胞/冰冻量的医院**院级贡献毛利率被系统性拉虚高**。
- 原文档只堵一半（不变量⑧ 仅剔 FISH/NGS/分子），细胞/冰冻照混进率。
- 与 Q4 同根：Q4="有 marker 但不是我们染的"（`lab_revenue=0`）；Q6="有 `lab_revenue` 但无 IHC 材料信号"（`lab_revenue>0` 且无 marker/特染）——纯细胞/冰冻技术活，P0 成本模型对它无效。

## 决策

**P0 精确贡献毛利率只覆盖 IHC/组织学线。准入闸（与 Q4 统一）：**
```
进"精确贡献毛利率"的 case ⟺ 有 IHC 材料信号(真抗体 marker 或 special_stain_count>0)  AND  lab_revenue>0
```
不满足者一律出率、进标注桶（不冒充精确、不拉虚高）：
- `lab_revenue=0`（有 marker）→ **诊断桶**（代阅片，Q4/ADR-002）。
- `lab_revenue>0` 但**无 IHC 材料信号** → **"成本模型不适用/待判"桶**：含**纯细胞病理/冰冻/宫颈液基技术线**（P0 无成本模型）**或**缺 marker 的 IHC 案。用 `classifier`（`case_revenue_lines.charge_code` 走逐院 config 分线）再判为 `非IHC线·成本未建模` / `needs_marker`。
- 院内 FISH/NGS 同此；外购转销 NGS 有已核外包成本另桶 `售价−外包成本`。

**透明度（硬要求）**：看板必出 **`P0率覆盖技术收入占比`**（= 进率的 IHC 线实收 ÷ 全部技术实收）。80% 是细胞的院此值低 → 明示"IHC 率 ≠ 全院毛利率"，防误读。

**为什么不 gate 在 `business_line`**：它和 `service_step_scope` 一样是预埋列、无写入路径、全空 → gate 上去=Q3 死循环。改用**"有无 IHC 材料信号"**当准入闸（已计算的数据），线的细分用 `classifier`（本就在算 `lab_revenue` 的同一逐院 config）事后判，不阻断出数。

## 后果

- ✅ 贡献毛利率成为**真正的 IHC/组织学线率**，不被细胞/冰冻/宫颈/分子拉虚高。
- ✅ Q4+Q6 统一成**一个准入闸**（有 IHC 材料信号 AND `lab_revenue>0`），实现简单、今天可产。
- ✅ 不 gate 在没人填的 `business_line` 上，无 Q3 死循环。
- ⚠️ 保守 miss：真 IHC 案但 marker 缺失会暂入待判桶（标 `needs_marker`，非计入虚高率）——诚实保守，非误算。
- ⚠️ 细胞/冰冻线的成本模型 = 未建模（这些线的成本口径是**后续独立课题**，非 P0 内圈）。

## 备选（未采纳）
- 只剔分子/NGS（原不变量⑧）：否——漏了细胞/冰冻/宫颈，率照样虚高。
- gate 在 `business_line`：否——预埋未填，Q3 死循环。
- 给细胞/冰冻现场建成本模型：否——超出 P0 内圈、无数据源，属后续。
