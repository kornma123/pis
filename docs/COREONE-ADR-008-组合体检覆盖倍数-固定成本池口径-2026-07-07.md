# ADR-008 · 组合体检「覆盖倍数」的固定成本池口径

> **日期**：2026-07-07 | **状态**：已采纳（PM `/grill-with-docs` 逐分支拷问·D1/D2/D5/D13/D18） | **性质**：hard-to-reverse（决定第 1 层组合体检主指标的分母来源与呈现·四轮外审收敛框架的地基之一）。
> **关联**：`~/Desktop/P0决策逻辑-四轮收敛终稿.md`（两层框架·覆盖倍数）；`portfolio-health.ts`（`buildPortfolioHealth`/`fixedRecoveryRate`）；`~/Desktop/非P0域-修复方案.md` §C（常量冻结·趋势打标）；`/CONTEXT.md`「固定成本池 / 覆盖倍数」；`docs/mockups/hospital-cm-两层框架-mockup.html`。

## 背景（grill 逐分支核实）

- 四轮外审收敛终稿把「全组合总贡献毛利 ÷ 固定成本池 = 覆盖倍数」升为**第 1 层组合体检主指标**。
- 但「固定成本池」**不是 P0 三件套里的数据**（P0 spec §1 只用 case_revenue/lis_case_markers/lis_cases/约定价），代码 `buildPortfolioHealth` 现只把它当**手填参数**（`fixedPool` query·默认 0），无来源、无口径、无变更控制。
- 覆盖倍数「只看趋势·不信绝对值」（校准前），但**趋势要历史**、**分母要月月一致**才有可比性——一个无来源、可随手改的分母，会让主指标要么空、要么被悄改污染。

## 决策

**固定成本池 = 财务手工维护的月度配置，只放固定开销、系统约束口径、只整盘不摊单院、变更留痕并在趋势线打标；覆盖倍数校准前只看趋势，且不是 mockup 的头号数——∑贡献毛利（真值）才是。**

**1｜来源 = 财务手工月度配置（D1）**：新建一张 config 表（一个数/月 + 留痕），财务维护。**不**从 ABC/成本引擎自动汇总（会把 P0 贡献毛利口径与 ABC 全成本口径耦合·ADR-003 刻意分开·且 ABC 数据是 seed）。

**2｜口径 = 只放固定开销·系统约束（D2·双计护栏）**：池 = 在编人力工资 + 设备折旧 + 房租 + 间接（真正不随片数变的）。**明确排除任何材料/可变成本**——材料已在贡献毛利里减过一次，再放进池 = 双罚。config 表字段注释 + UI 旁注把这条口径写死，防财务误填「月度总成本」。可变人力（外包/加班/计件）**既不进 CM 也不进池**，仍是 CM 已披露的「未含可变人力」乐观上界缺口。

**3｜只整盘、绝不摊单院（D5·死亡螺旋护栏·最关键）**：固定成本池**只在整盘体检当分母**；对照表/去留信号**绝不**把它摊到单个医院。按「这家医院该摊多少固定成本 → 单院全成本盈亏」算，就是四轮要杀的死亡螺旋（顶梁柱被摊亏 → 砍 → 固定成本更没人分）。单院只看：绝对贡献毛利 + 率 + 趋势（已减可避免材料·不摊固定）。〔与 ADR-003 一致：ABC 全成本那半也不喂去留。产能费 §4a 的**逐账户产能费**是**处方层**（目标报价）专用、且门控在产能实测后，不是体检/去留的单院固定摊派。〕

**4｜变更留痕 + 趋势线打标（D13·照 §C 范式）**：池每次变更留痕（谁·何时·从多少改到多少·为什么）+ 在趋势线对应月打标注。让「池变了」（年度调薪/加设备/房租涨）和「业务变了」分得开——否则手填分母悄改，覆盖倍数跳变会被当业务波动误读。这正是修复方案 §C（常量冻结·趋势打标）防的坑：恒定常量不破坏趋势可比性，真正致命的是中途被悄改无标注。

**5｜覆盖倍数不是头号数（D18·呈现层）**：覆盖倍数堆了一堆前提（手填池·只看趋势·要 3 个月历史·池变打标）= 页上最不牢的数。故 mockup **hero = ∑贡献毛利（真算出来的值）+ 趋势**；覆盖倍数**降为次级**、带「只看趋势·校准前」标。hero 应是最可信的数，不是最会误导精确的数。〔框架不变——覆盖倍数仍是第 1 层的概念指标，只是呈现上不冒充精确 hero。〕

## 后果

- ✅ 第 1 层主指标有了明确、可维护、可审计的分母来源，且不与 ABC 口径耦合。
- ✅ 双计护栏（排材料）防覆盖倍数把材料罚两次；整盘-only 护栏把死亡螺旋挡在体检层外。
- ✅ 变更打标护住「只看趋势」的地基（分母悄改不再被当业务波动）。
- ✅ hero 换成真值 ∑CM → 更诚实、adoption 更稳（财务不会把最不牢的数当结论）。
- ⚠️ 待办（本 ADR 只定口径·实现另立）：固定成本池 config 表 + 留痕 + 趋势打标的后端/前端；覆盖倍数绝对值判断仍待标准成本校准里程碑。
- ⚠️ 影子模式仍在（`PORTFOLIO_HEALTH_GATES_VERIFIED=false`）：三门 A/B/C 未验收前，本层输出不进经营研判。
