# COREONE 成本域文档 · 权威索引（一页看清「跟哪个 / 别跟哪个」）

> **日期**：2026-07-06 | **性质**：导航索引（不含新口径，只指路）。
> **为什么有这页**：成本域曾**反复推翻旧方案**（按医院全成本盈利 → G2 全成本毛利 → 逐抗体"算全" → 最终收敛到 **P0 院级贡献毛利·标准成本**）。旧文档没删（留历史/仍有现行节），散在 `docs/` 里，新来的实现者很容易**照着一份过时口径开工**。这页是**唯一入口**：先看这里认清当前权威，再动手。
> **当前权威一句话**：院级成本口径 = **贡献毛利（标准成本口径）= 实收（`lab_revenue`）− 白名单内可避免材料成本**；**工时/设备/房租/在编人力/对照片永不进成本**。经营呈现采用两层框架：组合层看覆盖倍数/产能利用率且不点名账户；账户层把绝对贡献额、率、同账户趋势并列给人判断，系统不自动排名、打分或生成砍院/谈价清单。ABC 全成本只在同一条瀑布内作长期趋势参考，不驱动具名账户决策信号。（权威定义见下 §1 的 `CONTEXT.md` + P0 spec + ADR-008。）
>
> **怎么用这页**：① 要实现/改成本口径 → 只读 §1 的文件。② 翻到一份 2026-06/07-02 的旧成本文档、拿不准 → 来这页查它在 §2 的状态（PARTIAL/SUPERSEDED）+ 哪节过时、哪节仍现行。③ 碰到 LIS 计费 / FRS-14 出入库成本报表 / BOM 消耗对账 → §3，那是**别的域**，别混。④ 想复现黄金锚 / 查缺价抗体 / 弱锚校准 / 手核脚本 → §4。
> **状态口径**：本页各文件的 PARTIAL/SUPERSEDED 标注**镜像自各文件顶部自己贴的头**（2026-07-06 由 PR #71/#72 贴）；若某文件头与本页不一致，**以该文件头为准**并回来修本页（防两处漂移）。**此规则同样适用于 §1 的正向权威断言**（含下方 ADR 明细表的「已采纳/当前权威」列）——某 ADR 文件头若已贴 SUPERSEDED/PARTIAL，则 §1 对它的正向断言也**以文件头为准**、须回来同步（2026-07-13 补：防「§1 说现行、文件头说已废」的单向漂移）。

## §5 · 4 条关键矛盾速查（旧文档最容易照错的 4 处）

> 每条：**旧说法（在哪些文档）→ 现权威（哪份·哪节）→ 一句判据**。

1. **全成本摊到单片 vs 只减可避免材料**
   - 旧 ❌：单位技术成本 = 材料 + **工时 + 设备+间接**，摊到片/例（G2 §1/§2、账实设计基线 §1.3「算全」）。
   - 现 ✅：贡献毛利 = 实收 − **白名单内可避免材料**（一抗/二抗显色/特染/组织处理料）；**工时/设备/房租/在编/对照片永不进成本**（`CONTEXT.md`「可避免材料成本」+ P0 spec §2 + ADR-004）。
   - 判据：一个成本量属①②③哪档？②档（固定/摊派）**不下沉、不进单片成本**。

2. **ABC 全成本利润砍院 vs 两层框架人工判断**
   - 旧 ❌：利润 = 实收 − ABC **全成本**，多级 CM 上卷，据此排名/砍院（按医院盈利 4 份、v1监测 §1.3、调研设计）。
   - 现 ✅：组合层看覆盖倍数/产能利用率且**不点名账户**；账户层将绝对贡献额、率、同账户趋势并列供人判断，系统**不自动排名、打分、生成砍院/谈价清单**（收官页 §B + ADR-008）。ABC 全成本只留在同一条瀑布的长期趋势参考层，**不喂具名账户红/排名/砍院/谈价**（ADR-003 保留部分）。
   - 判据：`/hospital-pnl` 不应由 ABC 或贡献毛利自动发具名账户去留/谈价信号；见到“按全成本毛利或贡献毛利率自动排名砍院/谈价”= 过时。

3. **`net_amount` vs `lab_revenue` + 同源闸**
   - 旧 ❌：用 `net_amount`（=lab+诊断+外送+未分配）或"实收×技术占比"估算当利润分子（调研设计、v1监测收入侧估算 lane）。
   - 现 ✅：分子 = **`case_revenue.lab_revenue` 直取**（且仅当 `lab_revenue IS NOT NULL` 且 `revenue_source∈(statement,corrected)`）；染色成本**只在 `lab_revenue>0` 时扣**=同源闸（`CONTEXT.md`「实收」+ ADR-002）。
   - 判据：分子不是 `net_amount`；`lab_revenue=0`（代阅片/纯诊断）→ 不扣染色成本、移出染色贡献毛利。

4. **库存/实际消耗对账 vs P0 不碰第四类源**
   - 旧 ❌：标准/实际**双口径盘点**+差异回摊、按实际消耗算利润（v1监测 §1.5/§9、账实设计早期 framing）。
   - 现 ✅：P0 是**标准成本口径**，**不引入任何第四类源**（inventory / outbound / batches / 消耗对账 / 实际消耗 / 财务实付 / 工时·设备台账）（P0 spec §1 框死 + §4 不变量⑤）。实际消耗留 ABC 中圈，不进 P0 贡献毛利。
   - 判据：P0 只用 §1 列的那几张表；看到"从库存/出库/实际消耗推单片成本"= 越界。

## 给 PM 的一段白话汇总

成本域这块过去反复推翻旧方案（先想按全成本给每家医院算利润、再想给每片切片摊全成本），最后定下来的是一个更稳的口径：**「贡献毛利」= 这家医院这个月实收的钱，减掉「多做一片就多花、我们自己做才会花」的材料钱**（试剂、显色、特染这类），**人工、设备、房租这些固定开销一律不摊进去**——因为摊了会把本来该留的医院算成"亏损"而误砍（死亡螺旋）。

这页文档不含任何新规则，它是一张**导航图**：告诉后面接手的人「现在到底该照哪份文档实现」（§1 那几份），以及「翻到 2026 年 6 月那批旧文档时，哪些段落已经作废、哪些还在用别删」（§2）。同时把两类**容易被误当成同一件事**的东西划清楚：LIS 计费是"收入"侧、FRS-14 是"耗材进出仓库"的成本报表，**都不是**"院级贡献毛利"这个域，别拿它们去搭医院盈利模型。目的只有一个：让"反复推翻"到此为止，后面不再有人照着过时口径重走一遍老路。
