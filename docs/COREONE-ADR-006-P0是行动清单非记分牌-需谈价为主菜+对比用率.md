# ADR-006 · P0 是「行动清单」不是「记分牌」——需谈价为主菜 + 院际对比用率

> **日期**：2026-07-04 | **状态**：已采纳（PM 拷问 Q9 #1+#2）| **性质**：hard-to-reverse（决定产品重心与看板默认视图 + 院际对比口径）。
> **关联**：P0 spec §5 行动层 / §4 不变量④；ADR-005（去留判定）；`[[adoption-first-design]]`。

## 背景（PM Q9 #1+#2）

- **#1 采纳空间没设计**：§5 输出裁决词（可留/需谈价/停止候选），但没字段回答**谁看 / 多久看 / 看完谁做什么**。裁决没有行动主人和动作 = 只是个数字。且**重心可能放错**：`停止候选`（砍院）戏剧但极罕见（临床/关系代价，一年可能零次）；`需谈价` 才是**天天能用的主菜**（"这家这条线边际为负/CM 低于目标 → 拿缺口去谈扣率/涨价"）。财务每月真正打开看的是"哪几家滑进需谈价、该带什么数字谈判"，不是砍院记分牌。
- **#2 每例 CM 院际对比被 case-mix 扭曲**：不变量④原用**每例贡献毛利**排名。但每例 CM 混了病例复杂度——A 院做简单单抗 case（每例 ¥X）、B 院做 10 抗大 case（每例 ¥10X）→ 每例看 B 甩 A，其实按每片/每抗可能一样。拿每例 CM 排名 = **偏爱做大 case 的院、误伤做简单便宜 case 但单位效率高的院**。

## 决策

**#1 P0 定位 = 行动清单；产品重心 = 需谈价（月度主菜）；默认视图 = 需谈价院清单 + 谈判弹药。**
- 每个状态绑定 **行动主人 / 复核节奏 / 建议动作**（见 §5 行动层表）。
- `需谈价` 携带**谈判弹药**字段：本院**哪条线/哪些抗体 CM 为负 + 缺口 ¥X + 当前扣率 vs 目标率差** → 财务直接拿去谈。
- `停止候选` 降为**罕见的季度/触发上会评估**（非自动砍、计临床/关系代价），不是看板头条。
- 看板默认视图 = "本月滑入需谈价的院 + 每家弹药"，不是砍院记分牌。

**#2 院际对比口径改复杂度中性指标。**
- 跨院对比**主用 贡献毛利率（CM÷lab_revenue）**、辅 **每片贡献毛利（CM÷片数）**——case-mix 中性。
- **每例贡献毛利禁用于跨院排名**（case-mix 敏感）→ 仅同院趋势；绝对贡献力度看**院级总贡献毛利**。
- 仍只比同口径院（含前处理 vs 仅染色不直接排名）。

## 后果

- ✅ P0 从"给个裁决数"变成"告诉财务这个月拿什么数字去找哪几家谈"——踩中日常价值、adoption 才成立。
- ✅ 重心从罕见的砍院挪到高频的谈价，产品被用起来。
- ✅ 院际对比不再误伤做简单便宜 case、单位效率高的院。
- ⚠️ 谈判弹药字段要能下钻到"线/抗体级 CM 缺口"——依赖 §10 的逐 marker/逐线成本可拆（已具备，实现工作量）。
- ⚠️ 目标率（`CM_TARGET`）是谈判弹药的基准，PM 须定（不同合作形态可不同）。

## 备选（未采纳）
- 只出裁决词不出行动/弹药：否（Q9#1 病根，没人拿它做事）。
- 头条放停止候选记分牌：否（罕见、戏剧、非日常价值）。
- 每例 CM 做院际排名：否（case-mix 扭曲，Q9#2）。
