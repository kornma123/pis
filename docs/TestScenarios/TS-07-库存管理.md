## 测试场景

| 场景ID | 场景描述 |
|--------|----------|
| INV-01 | Given 有库存数据的物料，When 调用 GET /api/v1/inventory，Then 返回 200，库存列表包含 materialId、code、name、stock、status、batch、expiry 字段 |
| INV-02 | Given 物料有多个入库批次，When 调用 GET /api/v1/inventory，Then 同物料+同批号+同库位合并为一条记录，stock 为聚合值 |
| INV-03 | Given 物料库存正常且有效期>30天，When 调用 GET /api/v1/inventory，Then 该记录 status="normal" |
| INV-04 | Given 物料库存<=minStock 且 minStock>0，When 调用 GET /api/v1/inventory，Then 该记录 status="low-stock" |
| INV-05 | Given 物料有效期<=today+30天且>today，When 调用 GET /api/v1/inventory，Then 该记录 status="warning" |
| INV-06 | Given 物料有效期<=today，When 调用 GET /api/v1/inventory，Then 该记录 status="expired" |
| INV-07 | Given 物料库存<=0，When 调用 GET /api/v1/inventory，Then 该记录 status="out-of-stock" |
| INV-08 | Given 按分类筛选 categoryId="试剂类ID"，When 调用 GET /api/v1/inventory?categoryId=xxx，Then 仅返回该分类下的库存记录 |
| INV-09 | Given 按库位筛选 locationId="库位ID"，When 调用 GET /api/v1/inventory?locationId=xxx，Then 仅返回该库位下的库存记录 |
| INV-10 | Given 低库存筛选 status="low-stock"，When 调用 GET /api/v1/inventory?status=low-stock，Then 仅返回 status 为 low-stock 的记录 |
| INV-11 | Given 搜索库存 keyword="苏木素"，When 调用 GET /api/v1/inventory?keyword=苏木素，Then 返回名称或编码包含"苏木素"的库存记录 |
| INV-12 | Given 调用 GET /api/v1/inventory/stats，When 返回统计看板，Then 包含 totalMaterials、totalStockValue、totalStockCount、normalCount、lowStockCount、expiringCount、expiredCount |
| INV-13 | Given 库存统计看板，When 检查各状态计数，Then normalCount + lowStockCount + expiringCount + expiredCount ≈ totalStockCount（有库存的批次数） |
| INV-14 | Given 入库 100 单位某物料，When 调用 GET /api/v1/inventory，Then 该物料 stock 增加 100 |
| INV-15 | Given 出库 30 单位某物料，When 调用 GET /api/v1/inventory，Then 该物料 stock 减少 30 |
| INV-16 | Given 库存计算公式验证，When 统计某物料库存，Then 库存 = SUM(inbound.quantity WHERE status=completed AND is_deleted=0) - 已出库数量 |
| INV-17 | Given warehouse_manager 用户，When 调用 GET /api/v1/inventory，Then 返回 200，正常读取 |
| INV-18 | Given technician 用户，When 调用 GET /api/v1/inventory，Then 返回 200，正常读取 |
| INV-19 | Given pathologist 用户，When 调用 GET /api/v1/inventory，Then 返回 200，正常读取 |
| INV-20 | Given procurement 用户，When 调用 GET /api/v1/inventory，Then 返回 200，正常读取 |
| INV-21 | Given finance 用户，When 调用 GET /api/v1/inventory，Then 返回 403 Forbidden |
| INV-22 | Given 无 Token 请求，When 调用 GET /api/v1/inventory，Then 返回 401 Unauthorized |
| INV-23 | Given 过期 Token，When 调用 GET /api/v1/inventory/stats，Then 返回 401 Unauthorized |
| INV-24 | Given 伪造 Token，When 调用 GET /api/v1/inventory，Then 返回 401 Unauthorized |
| INV-25 | Given 分页 page=0，When 调用 GET /api/v1/inventory?page=0，Then page=0 被修正为 1 |
| INV-26 | Given 分页 pageSize=1000，When 调用 GET /api/v1/inventory?pageSize=1000，Then 返回最多 1000 条记录 |
| INV-27 | Given 搜索空字符串 keyword=""，When 调用 GET /api/v1/inventory?keyword=""，Then 返回全部库存记录 |
| INV-28 | Given 响应结构检查，When 调用 GET /api/v1/inventory，Then 每条记录包含 id（INV-${material_id}-${batch_no} 格式）、availableStock（=stock） |
| INV-29 | Given 某物料无库存，When 调用 GET /api/v1/inventory，Then 该物料可能不显示（取决于聚合结果是否包含 0 库存） |
| INV-30 | Given 有效期字段 expiry，When 检查库存记录，Then expiry 为该批次入库记录中的最大有效期 |
