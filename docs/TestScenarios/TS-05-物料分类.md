## 测试场景

| 场景ID | 场景描述 |
|--------|----------|
| CAT-01 | Given 任意角色用户已登录，When 调用 GET /api/v1/categories/tree，Then 返回 200，完整三级树形结构，包含 children、isLeaf、count 字段 |
| CAT-02 | Given admin 用户，When 调用 POST /api/v1/categories 传入 name="新一级分类"、level=1，Then 返回 201，code 为 MAX+100（如 500） |
| CAT-03 | Given admin 用户，When 调用 POST /api/v1/categories 传入 name="新二级分类"、level=2、parentId="一级分类ID"，Then 返回 201，code 为同 parent 下 MAX+1 |
| CAT-04 | Given admin 用户，When 调用 POST /api/v1/categories 传入 name="新三级分类"、level=3、parentId="二级分类ID"，Then 返回 201，code 为同 parent 下 MAX+1 |
| CAT-05 | Given admin 用户编辑分类时传入 name="修改后名称"，When 调用 PUT /api/v1/categories/:id，Then 返回 200，仅 name 更新 |
| CAT-06 | Given admin 用户删除一个无子分类、无物料关联的三级分类，When 调用 DELETE /api/v1/categories/:id，Then 返回 200，逻辑删除 |
| CAT-07 | Given 任意角色用户，When 调用 GET /api/v1/categories，Then 返回 200，分页列表包含 id、code、name、level、parentId |
| CAT-08 | Given 搜索分类 keyword="免疫组化"，When 调用 GET /api/v1/categories?keyword=免疫组化，Then 返回匹配的免疫组化分类 |
| CAT-09 | Given 删除有子分类的一级分类，When 调用 DELETE /api/v1/categories/:id，Then 返回 409 "Has children" |
| CAT-10 | Given 删除有关联物料的三级分类，When 调用 DELETE /api/v1/categories/:id，Then 返回 409 "Has materials" |
| CAT-11 | Given 创建分类时未传 name 字段，When 调用 POST /api/v1/categories，Then 返回 400 "Name and level required" |
| CAT-12 | Given 创建分类时未传 level 字段，When 调用 POST /api/v1/categories，Then 返回 400 "Name and level required" |
| CAT-13 | Given technician 用户，When 调用 GET /api/v1/categories/tree，Then 返回 200，正常读取 |
| CAT-14 | Given procurement 用户，When 调用 GET /api/v1/categories，Then 返回 200，正常读取 |
| CAT-15 | Given technician 用户，When 调用 POST /api/v1/categories，Then 返回 403 Forbidden |
| CAT-16 | Given 分类树响应，When 检查一级分类节点，Then code 以 00 结尾（如 100、200、300、400） |
| CAT-17 | Given 分类树响应，When 检查三级分类节点，Then isLeaf=true，count 为该分类下直接关联的物料数量 |
| CAT-18 | Given 创建分类时传入 level=4（非法值），When 调用 POST，Then 后端未严格校验 level 合法性（可能创建成功或报错） |
| CAT-19 | Given 无 Token 请求，When 调用 GET /api/v1/categories/tree，Then 返回 401 Unauthorized |
| CAT-20 | Given 过期 Token，When 调用 GET /api/v1/categories，Then 返回 401 Unauthorized |
| CAT-21 | Given SQL 注入 payload（name="' OR '1'='1"），When 调用 POST /api/v1/categories，Then 创建成功，无 SQL 注入漏洞 |
| CAT-22 | Given XSS payload（name="<script>alert(1)</script>"），When 调用 POST /api/v1/categories，Then 创建成功，name 原样存储 |
| CAT-23 | Given 分页 page=0，When 调用 GET /api/v1/categories?page=0，Then page=0 被修正为 1 |
| CAT-24 | Given 搜索空字符串 keyword=""，When 调用 GET /api/v1/categories?keyword=""，Then 返回全部分类 |
| CAT-25 | Given 创建一级分类后，When 查询分类树，Then 新分类出现在树形结构中，children 为空数组 |
| CAT-26 | Given 分类树中 count=0 的叶子节点，When admin 调用 DELETE，Then 返回 200，删除成功 |
| CAT-27 | Given 编辑分类时传入 code 字段，When 调用 PUT，Then code 不被更新（接口不支持修改 code） |
| CAT-28 | Given 编辑分类时传入 parentId 字段，When 调用 PUT，Then parentId 不被更新（接口不支持调整父分类） |
| CAT-29 | Given 创建分类时 code 冲突，When 调用 POST，Then 返回 409 "Code already exists" |
| CAT-30 | Given 三级分类树响应，When 检查二级分类节点，Then level=2，code 在 101-199 范围内 |
