| 场景ID | 场景描述 |
|--------|----------|

| PROJ-02 | Given 创建项目时传入 code="PROJ-001"、name="HER2检测"、type="ihc"，When 调用 POST，Then 返回 201，所有字段正确存储 |
| PROJ-03 | Given admin 用户编辑项目时传入 name="修改后名称"，When 调用 PUT /api/v1/projects/:id，Then 返回 200，name 更新成功 |
| PROJ-04 | Given admin 用户删除项目，When 调用 DELETE /api/v1/projects/:id，Then 返回 200，逻辑删除 |
| PROJ-05 | Given technician 用户，When 调用 GET /api/v1/projects，Then 返回 200，正常读取 |
| PROJ-06 | Given pathologist 用户，When 调用 GET /api/v1/projects，Then 返回 200，正常读取 |
| PROJ-07 | Given pathologist 用户，When 调用 POST /api/v1/projects，Then 返回 403 Forbidden |
| PROJ-08 | Given warehouse_manager 用户，When 调用 GET /api/v1/projects，Then 返回 403 Forbidden |
| PROJ-09 | Given 项目详情查询，When 调用 GET /api/v1/projects/:id，Then 返回 200，包含 costStats（totalCost、sampleCount、unitCost） |
| PROJ-10 | Given 项目关联出库单后，When 查询项目详情，Then costStats.totalCost 包含该项目的所有出库成本之和 |
| PROJ-11 | Given 项目无出库记录，When 查询项目详情，Then costStats.totalCost=0，sampleCount=0，unitCost=0 |
| PROJ-12 | Given 按类型筛选 type="ihc"，When 调用 GET /api/v1/projects?type=ihc，Then 仅返回免疫组化类型的项目 |
| PROJ-13 | Given 按状态筛选 status="active"，When 调用 GET /api/v1/projects?status=active，Then 仅返回 active 状态的项目 |
| PROJ-14 | Given 搜索项目 keyword="HER2"，When 调用 GET /api/v1/projects?keyword=HER2，Then 返回名称或编码包含"HER2"的项目 |
| PROJ-15 | Given 创建项目时 code 已存在，When 调用 POST，Then 返回 409 "Code exists" |
| PROJ-16 | Given 创建项目时未传 code 字段，When 调用 POST，Then 返回 400 "Code, name and type required" |
| PROJ-17 | Given 创建项目时未传 name 字段，When 调用 POST，Then 返回 400 "Code, name and type required" |
| PROJ-18 | Given 创建项目时未传 type 字段，When 调用 POST，Then 返回 400 "Code, name and type required" |
| PROJ-19 | Given 删除项目后，When 查询历史出库记录，Then 出库记录仍存在，project_id 成为悬空引用 |
| PROJ-20 | Given 无 Token 请求，When 调用 GET /api/v1/projects，Then 返回 401 Unauthorized |
| PROJ-21 | Given 过期 Token，When 调用 POST /api/v1/projects，Then 返回 401 Unauthorized |
| PROJ-22 | Given SQL 注入 payload（code="' OR '1'='1"），When 调用 POST，Then 创建成功，无 SQL 注入漏洞 |
| PROJ-23 | Given XSS payload（name="<script>alert(1)</script>"），When 调用 POST，Then 创建成功，name 原样存储 |
| PROJ-24 | Given 分页 page=0，When 调用 GET /api/v1/projects?page=0，Then page=0 被修正为 1 |
| PROJ-25 | Given 编辑项目 status="inactive"，When 调用 PUT，Then 返回 200，status 变为 "inactive" |
| PROJ-26 | Given 项目成本统计验证，When 创建 2 笔出库（成本分别为 100 和 200），Then costStats.totalCost=300，sampleCount=2，unitCost=150 |
| PROJ-27 | Given 搜索空字符串 keyword=""，When 调用 GET /api/v1/projects?keyword=""，Then 返回全部项目 |
| PROJ-28 | Given 响应结构检查，When 调用 GET /api/v1/projects，Then 每条记录包含 id、code、name、type、cycle、bomId、supportableSamples、status、manager |
| PROJ-29 | Given 项目详情中 costStats，When 检查返回结构，Then 包含 totalCost、sampleCount、unitCost 三个字段 |
| PROJ-30 | Given 创建项目时传入 cycle="3天"、manager="张医生"，When 调用 POST，Then 返回 201，cycle 和 manager 正确存储 |
