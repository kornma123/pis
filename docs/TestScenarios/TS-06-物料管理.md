## 测试场景

| 场景ID | 场景描述 |
|--------|----------|
| MAT-01 | Given admin 用户已登录，When 调用 GET /api/v1/materials，Then 返回 200，物料列表包含 code、name、stock、price、status 字段 |
| MAT-02 | Given admin 用户，When 调用 POST /api/v1/materials 传入 name="测试物料"、unit="ml"、categoryId="试剂类ID"，Then 返回 201，code 自动生成（如 REA-xxxxx），同时创建 inventory 记录（stock=0） |
| MAT-03 | Given admin 用户创建物料时传入自定义 code="REA-99999"，When 调用 POST，Then 返回 201，使用传入 code |
| MAT-04 | Given admin 用户，When 调用 GET /api/v1/materials/:id，Then 返回 200，详情包含 batches（按 expiry_date 升序）和 stockLogs（最近 20 条） |
| MAT-05 | Given admin 用户编辑物料时传入 price=99.99，When 调用 PUT /api/v1/materials/:id，Then 返回 200，price 更新成功 |
| MAT-06 | Given admin 用户删除一个 stock=0 的物料，When 调用 DELETE /api/v1/materials/:id，Then 返回 200，逻辑删除 |
| MAT-07 | Given warehouse_manager 用户，When 调用 GET /api/v1/materials，Then 返回 200，正常读取 |
| MAT-08 | Given technician 用户，When 调用 GET /api/v1/materials，Then 返回 200，正常读取 |
| MAT-09 | Given technician 用户，When 调用 POST /api/v1/materials，Then 返回 403 Forbidden |
| MAT-10 | Given 创建物料时未传 name 字段，When 调用 POST /api/v1/materials，Then 返回 400 "Name, unit and category required" |
| MAT-11 | Given 创建物料时未传 unit 字段，When 调用 POST /api/v1/materials，Then 返回 400 "Name, unit and category required" |
| MAT-12 | Given 创建物料时未传 categoryId 字段，When 调用 POST /api/v1/materials，Then 返回 400 "Name, unit and category required" |
| MAT-13 | Given 创建物料时传入重复 code="REA-00001"，When 调用 POST，Then 返回 409 "Code already exists" |
| MAT-14 | Given 删除 stock>0 的物料，When 调用 DELETE /api/v1/materials/:id，Then 返回 409 "Stock exists" |
| MAT-15 | Given 按分类筛选 categoryId="试剂类ID"，When 调用 GET /api/v1/materials?categoryId=xxx，Then 仅返回该分类下的物料 |
| MAT-16 | Given 按供应商筛选 supplierId="供应商ID"，When 调用 GET /api/v1/materials?supplierId=xxx，Then 仅返回该供应商的物料 |
| MAT-17 | Given 搜索物料 keyword="苏木素"，When 调用 GET /api/v1/materials?keyword=苏木素，Then 返回名称或编码包含"苏木素"的物料 |
| MAT-18 | Given 创建物料时传入 price=0，When 调用 POST，Then 返回 201，创建成功（price=0 合法） |
| MAT-19 | Given 创建物料时传入 price=-10（负数），When 调用 POST，Then 后端未显式拦截，可能创建成功（前端应限制） |
| MAT-20 | Given 批量禁用物料，When 调用 PATCH /api/v1/materials/batch-status 传入 ids=["id1","id2"]、status="inactive"，Then 返回 200，所有指定物料 status 变为 "inactive" |
| MAT-21 | Given 批量禁用时空数组 ids=[]，When 调用 PATCH，Then 返回 400 或不影响任何数据 |
| MAT-22 | Given 物料详情中 batches 列表，When 检查批次顺序，Then 按 expiry_date ASC 排列，最早过期的批次在前 |
| MAT-23 | Given 物料详情中 stockLogs，When 检查记录数，Then 最多返回 20 条 |
| MAT-24 | Given 创建物料后，When 查询 inventory 表，Then 存在一条记录，material_id=新物料ID，stock=0 |
| MAT-25 | Given 编辑物料 categoryId，When 调用 PUT，Then categoryId 更新成功，但 code 前缀不会自动更新 |
| MAT-26 | Given 无 Token 请求，When 调用 GET /api/v1/materials，Then 返回 401 Unauthorized |
| MAT-27 | Given 过期 Token，When 调用 POST /api/v1/materials，Then 返回 401 Unauthorized |
| MAT-28 | Given SQL 注入 payload（name="' OR '1'='1"），When 调用 POST /api/v1/materials，Then 创建成功，无 SQL 注入漏洞 |
| MAT-29 | Given XSS payload（name="<script>alert(1)</script>"），When 调用 POST /api/v1/materials，Then 创建成功，name 原样存储 |
| MAT-30 | Given 分页 page=0，When 调用 GET /api/v1/materials?page=0，Then page=0 被修正为 1 |
