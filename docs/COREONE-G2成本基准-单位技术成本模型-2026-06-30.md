## 0. 背景铁律（决定这份文档的边界）

- **纯实验室 = 技术工序**（制片/染色的耗材 + 技师工时 + 设备折旧/间接）。**报告/诊断 = 病理医生 = 不算、不在本系统**（国标"病理诊断费"OUT；"病理标本处理费"+染色费 IN）。
- **监管悖论的解**：成本**不向被监管的实验室索取**——耗材单价用**真实采购价**（不可操纵），工作量**从对账单反推**（不是自报，见 §4）。实验室要造假得同时改采购台账和对账单，难度大得多。
- **实际 vs 基准 = 监管信号**：基准是"应该花多少"（BOM 标准 × 真实单价），实际是台账"花了多少"，差异当纠错信号，不是要消灭。

## 2. 建模公式（三段式，所有业务线通用）

```
单位技术成本 = 耗材成本            + 工时成本                + 设备折旧/间接
             （真实单价 × BOM用量）  （技师时薪 × 工序综合分钟）  （按占比系数或台账反推）
```

**全局锚**：
- 技师时薪（上海）：¥57–85/工时（月薪 10–15K ÷ 22天 ÷ 8h）。🟡 招聘数据，非康湾真实工资单。
- 成本结构占比（材料:人力:间接 = **26%:40%:34%**）：AJCP 2018 国外锚。🔴 套国内属弱假设，间接费系数最不确定，须用康湾真实工资/折旧/房租回归校准。间接 ≈ (耗材+人力) × 0.52。

## 4. 工作量从对账单来（闭环，破监管悖论的关键）

复核指出最大缺口是"月度工作量表"，**但工作量就在导入的对账单里**：每一行 IN 服务 = 一个工作量单位。

```
对账单 → 收入 = Σ(IN行结算)                 [真，已有]
       → 工作量 = 数 IN 行(按业务线类)         [同一张单免费 derive；注意"前八项"类捆绑行≈8抗体，需拆算]
成本   = Σ(工作量 × 本表单位成本)              [耗材真·工时估]
毛利   = 收入 − 成本
```

**对账边界（复核确认）**：康湾台账"出库金额"= **耗材消耗**，只对应本表**耗材段**，不含工时/间接（那是康湾工资/房租账，不在这张台账里）。→ **能对"耗材段"量级，不能对"总成本"。**

## 5. 交叉校验 + 监管信号

**已具备的硬校验点（无需工作量）**：康湾每月合计逐月自洽；IHC 占比 70% 稳定锚。
**反推校验示例**：康湾月均 IHC 出库 ≈ ¥45.8k（¥2,109,244 ÷ **46 有效月**，注：模型误写 42/¥50k）÷ 混合耗材单价 ~¥12 ≈ 3,800 抗体/月 ≈ 175/天 ≈ 30–45 病例/天 → 量级可信 ✅。

**三级信号**：带宽内 🟢 正常；超带宽 1–1.5× 🟡 关注（连续 2 月）；超 1.5× 🔴 排查。
**具体规则**：①某抗体每人份 > 历史中位×1.5 → 采购异常/规格错档；②反推工作量 > 产能上限 → 串账或单价低估；③大类占比骤变（如 IHC 骤降、常规囤货飙升）→ 大宗采购非真实消耗，剔除再核；④原液/即用错档 → 系统性偏 6 倍（算法级监管点）。

## 6. 能用 / 别信 / 做不到（复核裁决）

**✅ 能直接当监管基线**：抗体每人份表（分原液/即用档）；特染盒价÷次数；制片单件耗材价；康湾台账做对账分母 + IHC 70% 占比锚。

**⚠️ 别信 / 先补数据**：任何"设备+间接"绝对值（国外占比反推，承重墙，须本地校准）；特染/TCT/冰冻工时（无中文量化源）。

**❌ 目前做不到**：总成本对账（台账只有耗材、无工时真实值）；按院精拆成本（仅 11.7% 消耗带院标签，其中 59% 集中石门中医院一家；"基因检测待用"是用途占位非医院）。
