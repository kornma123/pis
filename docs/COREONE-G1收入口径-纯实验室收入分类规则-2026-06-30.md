## 0. 两条边界原则（用户确认）

1. **IN = 实验室做的技术工序**（制片：取材/脱水/包埋/切片/HE；染色：免疫组化/特殊染色）。**医生诊断、报告 = OUT**（国标"病理诊断费" vs "病理标本处理费"）。
2. **且仅限康湾自己实验室做的**——在**对方现场**做的（如冰冻现场服务）= **诊断桶（OUT）**。
   > 措辞修正（codex 09 · A 裁决）：现场服务**并非"不耗康湾成本"**——它有人工/差旅/机会成本；准确说法是**「非实验室技术工序成本」**（不进制片/染色技术成本口径）。若日后做服务利润分析，需另设「服务成本桶」，勿与实验室工序成本混。

> 一句话：**纯实验室收入 = 康湾实验室亲自完成的制片+染色技术部分。**

|---|---|---|

## 2. 拆分方法（逐病例，工作量两层 fallback）

老捆绑码按「国标处理费(每工作量单位) : 诊断费(每病例)」比例拆，**逐病例**：

```
制片份额(IN) = 处理费率 × 工作量 / ( 处理费率 × 工作量 + 诊断费105 )
诊断份额(诊断桶) = 1 − 制片份额
IN金额 = 病例内该线捆绑行结算总额 × 制片份额
```

- 处理费率：组织 ¥36、细胞 ¥75/玻片、冰冻按组织 ¥36 近似。诊断费 ¥105/病例（一病例判读一次）。
- **⭐ 工作量来源＝两层 fallback（codex 09 · MED-1，实现口径以此为准，勿只读"按标本数"）**：
  1. **组织检诊**：优先 **LIS 真蜡块数**（对账单 × LIS 按病理号 join；最准，一部位常切多蜡块）。
  2. **无 LIS 时降级**：用**账单数量/行数**（1 部位/标本/玻片 ≈ 1 行）作估算下限。
  3. **TCT / 冰冻**：目前 LIS 无该字段 → 用**账单数量**（诚实降级，见 §3 与 codex D 裁决）。
- **为什么按工作量**：处理随蜡块/标本数增加，诊断按病例固定 → 多蜡块病例制片占比自然更高（比固定比例准）。
- **精度差**：账单数量估算全月 25,772 vs LIS 真蜡块 27,870（差 ~2,098）。**W4 ¥7,118 是无 LIS 的账单数量估算，为下限次锚，不可当最终实现 golden**（golden = LIS 版 ¥27,870）。

| 口径 | 纯实验室 | 说明 |
|---|---:|---|
| 数量-忽略（每行=1标本） | 22,835 | 最粗 |
| 数量-aware（账单数量） | 25,772 | 中 |

**按业务线 IN 来源（诚实分账，codex 09 · MED-4；复现脚本已逐线打印）**：组织制片(LIS蜡块) ¥13,079 / 染色(整条IN) ¥11,648 / TCT(账单数量) ¥3,106 / 冰冻(账单数量) ¥37 = ¥27,870。**只有组织制片由 LIS 真蜡块精算**，TCT/冰冻仍是账单数量估。

**LIS 双核验（同期同院对账单 × LIS，通过）**：免疫组化 LIS **80** ≈ 对账单反算 **79**；特染 LIS **2** = 对账单 **2**；蜡块 LIS **260** ＞ 账单部位 ~150（一个部位切多个蜡块，故组织制片应按 LIS 蜡块拆，不是账单部位/数量）。

> **⚠️ "165 病例 100% 匹配" 的准确含义（codex 09 · D 裁决）**：指对账单 165 个病理号**都在 LIS 存在**，**不等于**所有业务线都由 LIS 精算——组织检诊病例均有蜡块（0 例退化为 0 蜡块），但 TCT/细胞学病例本就无组织蜡块（LIS blk=0 属正常），其制片走账单数量。**"病理号匹配" ≠ "该线工作量字段可用"，两者要分开看。**
>
> **月份归属（codex 09 · LOW-1）**：按**结算表月份**归属（本表期间 20260201–20260228 = 26.2），**不按行内登记日期**。原始明细行含跨月登记日期（2 月/3 月）属正常，不构成异常；LIS 按月 join 时以结算表月份为准，避免"3 月日期为何在 2 月表"的假异常。（本项目 fixture 已把登记日期列置空，见脱敏说明。）

> 组织制片份额（最终）= `36 × LIS蜡块数 / (36 × LIS蜡块数 + 105)`，**逐病例**（对账单 × LIS 按病理号 join）。W4 ¥7,118 是无 LIS 的账单数量估算，为下限次锚，非最终 golden。

## 4. 产品级影响（重要）

**原"实验室收入"口径（全月 ¥55,541）把医生诊断/报告/现场服务都算进去了，真·纯实验室仅 ¥27,870（占 50.2%）——高估约 2 倍。** 院级盈亏收入侧、现有看板的"实验室收入"都要按本口径重算。

> **⚠️ 最弱假设 / 诚实边界（codex 09 · B 裁决，也是用户"最担心 1"）**：拆分公式 `36×蜡块/(36×蜡块+105)` 本身守恒、稳定、可解释，但把**国标 36/105 比例套到和睦家溢价单价**（组织 ¥165/190/310/470 vs 国标 ¥141）本质是**政策分摊，不是"证明康湾真实制片价值就是这个比例"**。风险：若溢价来自双语/外籍/服务，公式**高估** IN；若来自更复杂标本处理，公式**低估** IN。**需配敏感性区间**（如诊断锚 105 →本地协商值时 IN 怎么变）。当前作为「最强数据锚下的选定分摊口径」使用，不作为事实成本/售价证明。

## 6. 与 G2 成本口径的一致性（闭环）

G2 成本也**只算制片/染色工序、不含诊断** → 收入只算制片、成本只算制片，**口径一致**。医生诊断在本系统**既无收入也无成本**。G3 毛利 = G1(制片+染色收入) − G2(制片+染色成本)，干净闭环。
