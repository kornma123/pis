## 第 0 层 · 范围与边界
- 系统只核算**实验室实物工序**（制片/染色）的**成本 + 收入**。诊断/报告/远程会诊/外送分子/共建分成 = **不在本看板**（另有系统管）。判据=有没有实物工序。
- **实验室实收 = 医院收费 × 扣率**（结算口径）。扣率逐院逐项、随合同变，是前置主数据。

## 第 1 层 · 口径（核心，实现红线）

### 1.1 成本 / 收入总口径
- **成本永远全算**（料真花了），**收入按财务认定**（认不认、能不能补）。
- 桥接锚点 = **物理工序单位**（蜡块 / 免疫组化片 / 特染片），不是收费码。同一 LIS 计量既驱动收入拆分、又驱动成本。

### 1.2 收入侧：老混合码怎么拆
- 老"制片+诊断"混合码（国标 2703 系列）**不能整条接 BOM**。逐病例按 **制片份额 = 处理费率×工作量 /（处理费率×工作量 + 诊断费105）** 拆：组织费率 ¥36/标本、细胞 ¥75/玻片、冰冻按 ¥36 近似；工作量优先取 **LIS 真蜡块**，缺则降级账单数量（标 partial）。
- 制片份额 → 实验室收入；`1−份额` → **诊断桶（零成本，属医生、不进实验室盈亏）**。
- **免疫组化/特染 整条计入实验室**（纯染色技术费；医生判读的钱在病例的"检查与诊断"码里、已随组织拆分划走 → 不重复算）。**不照搬美国 TC/PC 比例硬拆**（中国老码不分、按张数收笼统诊断费，套他国比例=假精确）。

### 1.3 成本侧：逐抗体（免疫组化）
- **必须逐抗体**（真实台账 192 种 ¥0.29~99.82、差 344 倍；原液/即用差 6 倍）——不能用一个均价。
- **"算全"每片成本 = 一抗（台账真价）+ 二抗/显色（台账真价，上机二抗测试条~¥15/片）+ 技师工时（G2 估）+ 设备折旧（G2 估）**。60 例实证毛利率 ~70%（真价一抗+二抗占 48%，G2 估工时+设备占 52%，设备最弱）。
- **每人份成本取台账"第 14 列（已换算的每人份价）"**，勿再除换算率（曾除重致数字离谱的坑）。原液/即用分档。
- **账单 = 物理片数（含重染、剔特染）**；"重染"多是同抗体做在多蜡块/病灶（各收各钱），成本收入天然对齐。
- **成本完整度分档标注**：精算（有 LIS 抗体清单，逐抗体真价）↔ 粗估（只知数量，全院用量加权均价 × 数量）。缺台账价的抗体行级标"成本缺 N 价·毛利待定"。
- 特染成本：**默认"盒价 ÷ 标称次数"**（康湾买成品盒，G2 真实盒价 Masson¥318/网状¥549/抗酸¥195）+ 可选"实际得率"吸收到期损耗；自配逐步藏"更精确"入口。全成本 ≈ ¥20/次（人力主导）。

### 1.4 账实核对（差异 → 认定）
- **差异 = 账单片数 vs LIS 物理片数**（纯计数，不依赖成本价）。前提：两边到齐 + **院名对齐** + **病理号匹配率**达标。
- **匹配率默认阈值**（可调）：院名没对上 → 先对齐、**不出差异**；病理号匹配 **≥95% 正常算 / 80–95% 算但标"匹配偏低（仅参考）" / <80% 整院"先查"不出差异结论**；**未匹配病例永远单列"算不了"，不混进差异**。
- 缺列的线（TCT/冰冻 LIS 无片数字段）→ 标"按账单认·不可核"，不并进可核差异。
- **系统初判**（线索非定论，财务终判）：同蜡块同抗体重复 = 疑似**返工**；跨蜡块同抗体 = **多病灶（各收各钱）**；账单 > 实际 = 疑似**计费项目用错**；晚于计费期 = 疑似**超期免费**。
- **6 个认定原因（财务点，唯一术语串）+ 后续动作**：

  | 认定原因 | 成本 | 收入 | 下家 |
  |---|---|---|---|
  | **漏收，需补收** | 算 | 应补收 | → 补收追踪（③）驱动补收 |
  | **返工重做（不计费）** | 算 | 不收 | 计入"返工"指标 |
  | **超期，免费做的** | 算 | 收不到 | 计入"免费诊断"指标 |
  | **计费项目用错** | — | 按正确项更正 | → **待外部更正**（回原收费系统改，系统只标记） |
  | **LIS 记录不全** | — | — | → **待补数据** |
  | **核对无误** | — | — | 了结 |

  - **补收 gate**：只有"漏收，需补收"驱动补收；返工/超期收不回、不驱动。
  - **补收回填**：补收单标记已补收 → 自动计入本月实收（往月的记本月，上月定版不动）。

### 1.5 关账
- **部分关账 + 挂起**：就绪的院先关、未就绪的挂起（不被最慢一家拖死）。
- **关账前置 = 复核完成**（该院差异全认定），**不是**仅"数据到齐"。
- **关账 = 定版**（不可逆）：按钮 → 二次确认（关哪些/挂起哪些/合计实收/不可逆提示）。
- **关账后**迟到账单/补收/抗体价补齐 → **只记次月调整、不改定版**（抗体价补齐只重算成本、收入定版不变）。
- "已确认实收" **只算复核完成的院**。

### 1.6 通用性
- 机制（复核 → 认定 → 下家 + 成本全算）通用所有线；但**"差异"定义按线不同**：按数量收费的线（免疫组化/特染/TCT）= 账单数 vs LIS 物理片；**组织学是捆绑码（一例一口价、无数量）→ 差异另定义**（待做，见未决 B1）。

## 第 4 层 · 交互（状态机 + 权限）
**院·月复核状态**：待复核 →(逐条认定完)→ 复核完成 →(整月一起)→ 已关账（定版）。反向：复核完成 →**重新打开**→ 待复核；已关账 →**反关账**（慎用，一般迟到/补收记次月即可）。
**补收单状态**：待补收 →(收到)→ 已补收 / →(收不回)→ 已放弃。反向：已补收→**改回待补收**；已放弃→**恢复待补收**。
**所有反向动作**：**单人可做，但必须填理由 + 记经手人 + 进操作记录**，管理员可事后抽查。
**权限**：**财务 / 管理员都可操作**（非只读）；**单人可做，不设审批、不分人**（内控从简），只保留**留痕兜底**（认定/关账/放弃/反向都记经手人）。
