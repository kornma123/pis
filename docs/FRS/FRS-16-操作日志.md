## 1. 功能概述

系统操作日志查询，记录用户的关键操作行为（登录、创建、编辑、删除等）。用于审计追踪和问题排查。

| 项目 | 说明 |
|------|------|
| **功能定位** | 操作审计，问题追溯 |
| **可访问角色** | `admin`/`finance`（读） |
| **RBAC 控制** | `requireRole('admin','finance')` |
| **数据特点** | 只读，系统初始化 13 条日志 |

## 3. 接口详情

### 3.1 GET /logs/operation — 操作日志

#### 3.1.1 请求参数（Query String）

| 字段 | 必填 | 类型 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | ❌ | integer | 1 | 页码 |
| `pageSize` | ❌ | integer | 20 | 每页条数 |
| `startDate` | ❌ | date | - | 开始日期 |
| `endDate` | ❌ | date | - | 结束日期 |
| `userId` | ❌ | string | - | 用户 ID 筛选 |

#### 3.1.2 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | UUID |
| `userId` | string | 用户 ID |
| `username` | string | 用户名 |
| `operation` | string | 操作类型 |
| `description` | string | 操作描述 |
| `ip` | string | IP 地址 |
| `userAgent` | string | 浏览器 UA |
| `createdAt` | datetime | 操作时间 |

#### 3.1.3 隐含规则显式化

| 规则 | 说明 |
|------|------|
| 日志写入方式 | 由后端路由手动写入（非自动中间件），部分操作可能未记录 |
| 当前数据量 | 系统仅 seed 数据中有 13 条操作日志 |
| 不支持删除 | 日志不支持删除/清理接口 |
| 无敏感数据脱敏 | IP 和 UA 原样记录，未脱敏处理 |
