## 项目概览

- **项目名称**：COREONE 实验室耗材管理系统
- **版本**：v1.1
- **模块数量**：8个业务模块
- **前端页面数量**：20个页面
- **技术栈**：
  - **前端**：React 19 + TypeScript + Vite + React Router v7 + Recharts
  - **后端**：Express + PostgreSQL + Node.js
  - **数据库**：PostgreSQL 15+ with pg_cron
  - **UI**：自定义组件库 (components/index.tsx)

## 边界场景处理

| 场景 | 处理方案 | 涉及页面 |
|------|----------|----------|
| 物料列表为空 | 显示空状态插画 + "去新增"按钮 | `InventoryList.tsx` |
| 入库记录为空 | 显示空状态提示 | `InboundManagement.tsx` |
| 出库库存不足 | Toast提示"库存不足"，阻止提交 | `OutboundManagement.tsx` |
| 删除被引用分类 | Toast提示"存在物料，无法删除" | `CategoryManagement.tsx` |
| 网络请求失败 | Toast错误提示，按钮恢复 | 所有异步操作 |
| API 500错误 | Alert错误提示，提供重试 | 所有页面 |
