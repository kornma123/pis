# COREONE 数据状态笛卡尔积 E2E 场景设计

> **四维度**: 物料类型 × 库存状态 × 批次状态 × 操作  
> **总组合空间**: 4 × 4 × 4 × 4 = 256  
> **排除规则**: ① 无批次时无法执行"按批次号精确出库"（但允许 batchNo=null 出库）；② 库存为0时出库/报废导致负库存；③ 满库时入库可能导致超库容；④ 有过期批次时 FIFO 会分配过期品（业务冲突，非不可能）  
> **目标**: ≥80 个有效 E2E 场景，全部用 Given-When-Then 描述

---

## 维度定义

| 维度 | 取值 | 说明 |
|:---|:---|:---|
| **物料类型** | IHC 抗体 | 需冷链（-20°C），有严格有效期，按瓶/支管理 |
| | HE 染色试剂 | 常温保存，有有效期，按瓶管理 |
| | 通用耗材 | 一次性手套/口罩等，无有效期，按盒/包管理 |
| | 危化品 | 易燃/腐蚀性，需特殊存储，双人双锁 |
| **库存状态** | 库存为 0 | inventory.stock = 0 |
| | 低于安全库存 | 0 < stock ≤ safety_stock |
| | 正常 | safety_stock < stock < max_stock |
| | 满库 | stock ≥ max_stock |
| **批次状态** | 无批次 | batches 表中无该物料记录 |
| | 单批次临期 | 1 个活跃批次，expiry_date ≤ today + 30 天 |
| | 多批次 FIFO 混合 | ≥2 个活跃批次，按 expiry_date 升序排列 |
| | 有过期批次 | 至少 1 个批次 expiry_date ≤ today |
| **操作** | 入库 | 创建 inbound 记录，创建/更新 batch |
| | 出库 | 创建 outbound 记录，按 FIFO 分配 batch |
| | 盘点 | 创建 stocktaking 记录，调整 inventory.stock |
| | 报废 | 创建 scrap 记录，扣减 inventory.stock |

## 一、入库操作（20 个场景）

| # | 物料类型 | 库存状态 | 批次状态 | 场景摘要 (Given-When-Then) |
|:---|:---|:---|:---|:---|
| IN-01 | IHC 抗体 | 库存为 0 | 无批次 | Given IHC 抗体（Ki-67）库存为 0 且无批次记录，When 仓库管理员执行采购入库 10 支（批号 K202601、有效期 2027-01-01、需冷链标记），Then 入库单创建成功，批次表新增记录，库存更新为 10，预警中心不触发临期预警 |
| IN-02 | IHC 抗体 | 低于安全库存 | 单批次临期 | Given IHC 抗体库存 2（安全库存 5），现存单批次 2026-06-01 过期（临期），When 采购入库 20 支（新批号 K202602、有效期 2028-01-01），Then 入库成功，新批次创建，库存更新为 22，临期预警仍存在（旧批次未处理） |
| IN-03 | IHC 抗体 | 正常 | 多批次 FIFO 混合 | Given IHC 抗体库存 50（正常），现有 3 个活跃批次按 FIFO 排列，When 采购入库 30 支（新批号 K202603、有效期 2028-06-01），Then 入库成功，新批次加入活跃列表，FIFO 顺序调整为 expiry_date 升序，库存更新为 80 |
| IN-04 | IHC 抗体 | 满库 | 有过期批次 | Given IHC 抗体库存 100（已达 max_stock=100），现存 1 个过期批次 2025-12-01，When 尝试采购入库 10 支，Then 系统提示"库位容量已满，请先处理过期批次或调整库位"，入库被阻止 |
| IN-05 | HE 染色试剂 | 库存为 0 | 无批次 | Given HE 苏木素染液库存为 0 且无批次，When 直接入库 5 瓶（批号 H202601、生产日期 2026-01-01、有效期 2027-01-01），Then 入库成功，库存更新为 5，批次表新增记录 |
| IN-06 | HE 染色试剂 | 低于安全库存 | 单批次临期 | Given HE 伊红染液库存 1（安全库存 3），单批次 2026-06-15 过期，When 采购入库 10 瓶（新批号 H202602、有效期 2027-12-31），Then 入库成功，库存更新为 11，旧批次临期预警与新批次正常状态并存 |
| IN-07 | HE 染色试剂 | 正常 | 多批次 FIFO 混合 | Given HE 染液库存 30（正常），存在 2 个活跃批次（H202501 有效期 2026-08-01、H202502 有效期 2027-02-01），When 退货入库 3 瓶（关联原出库单 OB-202601-001），Then 退货入库成功，库存更新为 33，不创建新批次（或按原批号累加） |
| IN-08 | HE 染色试剂 | 满库 | 无批次 | Given HE 染液库存 200（满库，max_stock=200），无批次记录，When 直接入库 1 瓶，Then 系统提示"超过最大库存限制，不可入库"，阻止提交 |
| IN-09 | 通用耗材 | 库存为 0 | 无批次 | Given 一次性手套库存为 0，无批次记录（通用耗材通常不管理批次），When 采购入库 1000 双，Then 入库成功，库存更新为 1000，不创建批次记录（或不填 batchNo） |
| IN-10 | 通用耗材 | 低于安全库存 | 单批次临期 | Given 一次性口罩库存 20（安全库存 50），单批次（临期概念对耗材不适用，按入库时间算），When 采购入库 500 个，Then 入库成功，库存更新为 520，系统不触发临期预警（耗材无有效期） |
| IN-11 | 通用耗材 | 正常 | 多批次 FIFO 混合 | Given 载玻片库存 500（正常），存在 3 个入库批次，When 采购入库 1000 盒，Then 入库成功，库存更新为 1500，新批次加入 FIFO 队列 |
| IN-12 | 通用耗材 | 满库 | 有过期批次 | Given 一次性吸管库存 9999（满库），存在历史批次记录，When 尝试入库 1 盒，Then 系统提示库位已满，阻止入库 |
| IN-13 | 危化品 | 库存为 0 | 无批次 | Given 甲醛固定液库存为 0，无批次，危化品库位空，When 采购入库 2 瓶（批号 W202601、双人验收），Then 入库成功，库存更新为 2，批次创建，操作日志记录双人验收信息 |
| IN-14 | 危化品 | 低于安全库存 | 单批次临期 | Given 二甲苯库存 1（安全库存 2），单批次 2026-06-01 过期，When 采购入库 5 瓶（新批号 W202602、有效期 2028-01-01），Then 入库成功，库存更新为 6，旧批次临期预警仍存在 |
| IN-15 | 危化品 | 正常 | 多批次 FIFO 混合 | Given 酒精库存 20（正常），存在 2 个活跃批次，When 采购入库 10 瓶，Then 入库成功，库存更新为 30，新批次加入 FIFO 队列 |
| IN-16 | 危化品 | 满库 | 有过期批次 | Given 盐酸库存 50（满库，max_stock=50），存在过期批次 2025-06-01，When 尝试入库 1 瓶，Then 系统阻止入库并提示"危化品库位已满，请先处理过期品" |
| IN-17 | IHC 抗体 | 正常 | 有过期批次 | Given IHC 抗体库存 30（正常），存在 1 个过期批次 2025-12-01 和 1 个正常批次 2027-06-01，When 采购入库 20 支（新批号 K202604），Then 入库成功，库存更新为 50，过期批次 status=2 不参与 FIFO，新批次正常加入 |
| IN-18 | HE 染色试剂 | 库存为 0 | 多批次 FIFO 混合 | Given HE 染液库存为 0（所有批次已耗尽），历史存在 2 个 status=0 的已用完批次，When 采购入库 10 瓶，Then 入库成功，库存更新为 10，新批次创建，历史已用完批次不影响 |
| IN-19 | 通用耗材 | 低于安全库存 | 无批次 | Given 实验室纸巾库存 5（安全库存 10），无批次记录，When 采购入库 100 包，Then 入库成功，库存更新为 105，不创建批次 |
| IN-20 | 危化品 | 正常 | 无批次 | Given 漂白剂库存 10（正常），无批次记录，When 采购入库 2 瓶（需双人审批），Then 入库成功，库存更新为 12，操作日志记录审批人信息 |

## 三、盘点操作（20 个场景）

| # | 物料类型 | 库存状态 | 批次状态 | 场景摘要 (Given-When-Then) |
|:---|:---|:---|:---|:---|
| STK-01 | IHC 抗体 | 库存为 0 | 无批次 | Given IHC 抗体系统库存为 0，无批次，When 仓库管理员盘点输入实盘 0，Then 差异=0，不调整库存，盘点单状态完成 |
| STK-02 | IHC 抗体 | 库存为 0 | 单批次临期 | Given IHC 抗体系统库存为 0，但存在单批次临期记录（remaining=0），When 盘点输入实盘 0，Then 差异=0，库存保持 0，批次状态不变 |
| STK-03 | IHC 抗体 | 低于安全库存 | 多批次 FIFO 混合 | Given IHC 抗体系统库存 2（安全库存 5），2 个批次（各 1 支），When 盘点输入实盘 3（发现漏记入库），Then 差异=+1，库存调整为 3，stock_logs 记录 adjust 类型流水 |
| STK-04 | IHC 抗体 | 正常 | 有过期批次 | Given IHC 抗体系统库存 30，存在过期批次 2025-12-01（5 支，status=2）和正常批次 25 支，When 盘点输入实盘 25（过期品已物理隔离），Then 差异=-5，库存调整为 25，过期批次不参与盘点调整 |
| STK-05 | IHC 抗体 | 满库 | 多批次 FIFO 混合 | Given IHC 抗体系统库存 100（满库），3 个批次，When 盘点输入实盘 98（发现 2 支破损），Then 差异=-2，库存调整为 98，触发低库存预警（98 < max_stock 但可能 > safety_stock） |
| STK-06 | HE 染色试剂 | 库存为 0 | 无批次 | Given HE 染液系统库存为 0，无批次，When 盘点输入实盘 0，Then 差异=0，不调整 |

| STK-08 | HE 染色试剂 | 正常 | 多批次 FIFO 混合 | Given HE 染液系统库存 30，2 个批次，When 盘点输入实盘 28（2 瓶破损），Then 差异=-2，库存调整为 28，stock_logs 记录 adjust 流水 |
| STK-09 | HE 染色试剂 | 满库 | 无批次 | Given HE 染液系统库存 200（满库），无批次，When 盘点输入实盘 200，Then 差异=0，不调整 |
| STK-10 | HE 染色试剂 | 正常 | 有过期批次 | Given HE 染液系统库存 40，存在过期批次 5 瓶，When 盘点输入实盘 35（过期品已报废移除），Then 差异=-5，库存调整为 35，过期批次 remaining 更新为 0 |
| STK-11 | 通用耗材 | 库存为 0 | 无批次 | Given 一次性手套系统库存为 0，无批次，When 盘点输入实盘 0，Then 差异=0 |

## 统计汇总

| 操作 | 场景数 | 说明 |
|:---|:---:|:---|
| 入库 (IN) | 20 | 覆盖 4 物料 × 4 库存 × 4 批次的核心组合，含危化品双人验收、满库拦截 |
| 出库 (OUT) | 25 | 覆盖 FIFO 分配、batchNo=null、库存不足、过期跳过、BOM 一键出库 |

## 不可能组合排除说明

以下组合被排除（标记为 N/A）：

| 组合 | 排除原因 |
|:---|:---|
| 任意物料 + 库存为 0 + 多批次 FIFO 混合 | 数据不一致（库存为 0 时活跃批次 remaining 必为 0，不构成"混合"） |
| 任意物料 + 库存为 0 + 有过期批次 | 过期批次 status=2 且 remaining=0，与库存为 0 状态重合，不独立成场景 |
| 通用耗材 + 单批次临期 | 通用耗材无有效期字段，"临期"概念不适用，转化为"单批次"即可 |
| 通用耗材 + 有过期批次 | 通用耗材无有效期字段，"过期"概念不适用，转化为"多批次"即可 |

> **注**: 上述排除仅针对"批次状态"维度中有效期概念对耗材的不适用性。在实际矩阵中，通用耗材的"单批次临期"和"有过期批次"被替换为"单批次"和"多批次"的纯数量管理视角，仍保留有效场景。
