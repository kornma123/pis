## 关键上下文

### 技术约束
- `usePagination` 期望后端返回 `{ list: T[], pagination?: { total, page, pageSize } }`
- 若后端分页格式不同，需调整 `usePagination` 或后端统一格式
- `useUrlParams` 使用 `window.history.replaceState`，不触发页面刷新

### 组件使用方式
```tsx
// ConfirmDialog
<ConfirmDialog
  open={isOpen}
  title="确认删除"
  description="此操作不可撤销"
  confirmText="删除"
  confirmVariant="danger"
  onConfirm={handleDelete}
  onCancel={() => setIsOpen(false)}
/>

// Pagination
<Pagination
  page={page}
  pageSize={pageSize}
  total={total}
  onChangePage={setPage}
  onChangePageSize={setPageSize}
/>
```
