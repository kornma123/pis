# 前置问题处理方案

## 1. 编码自动生成规则

### 分类编码规则
病理实验室物料通常分为以下几大类，采用层级编码：

```
100  试剂
  101  分子诊断试剂
  102  病理技术试剂
  103  免疫组化试剂
  104  特殊染色试剂
200  耗材
  201  防护用品
  202  消毒用品
  203  通用耗材
300  设备配件
400  危化品
```

- 采用 `三位层级码`，一级 `100`、`200`，二级 `101`、`102`
- 新增分类时，同层级自增+1

### 物料编码规则
```
REA-00001   试剂
CON-00001   耗材
DEV-00001   设备配件
HZP-00001   危化品
```

- 前缀取自分类上级缩写（试剂 REA，耗材 CON）
- 序号5位自增
- 新增物料时自动生成，可手动修改

## 2. 库位多层级配置

### 病理实验室典型库位结构

```
A区 试剂冷藏区
  A01  4℃冷藏冰箱
    A01-01  冰箱上层
    A01-02  冰箱下层
  A02  -20℃冷冻冰箱
  A03  -80℃超低温冰箱
B区 常温耗材区
  B01  通用耗材货架
    B01-01  货架第一层
    B01-02  货架第二层
  B02  防护用品柜
C区 设备配件区
D区 危化品柜
E区 操作台暂存区
```

### 方案

1. **locations 表** 增加 `type` 字段（区/冰箱/货架/柜/操作台）和 `parent_id` 自关联
2. **前端** 库位选择改为级联选择：选区 → 选冰箱/货架 → 选层/格
3. **入库弹窗** 库位选择需展示层级路径

## 3. 参考价与实际入库价的意义

| 价格类型 | 定义 | 作用 |
|---------|------|------|
| **参考价** | 物料建档时预设的价格 | 采购预算、成本预估、审批依据 |
| **实际入库价** | 每次入库的真实价格 | 成本核算、价格波动分析 |

- **偏差分析**：系统自动对比实际价 vs 参考价，偏差>20%预警
- **成本归集**：出库成本按实际入库价加权平均计算
- **优化建议**：参考价根据实际入库价周期性（季度）自动调整

## 4. 安全库存/保险库存单位和交互

### 当前问题
- 前端只显示数字输入框，用户不清楚单位是什么

### 方案
- **单位**：与物料主单位一致（如 `盒`、`瓶`、`片`）
- **交互优化**：
  - 输入框右侧显示单位后缀 `安全库存  ___  盒`
  - 增加滑块/步进器交互
  - 提供参考值提示（如"参考日均消耗5盒，建议安全库存设置30盒"）

## 6. 入库弹窗报错重复提交

### 根因分析
- 前端点击"确认入库"按钮后，后端处理超时但部分操作已执行
- 用户再次点击，生成重复入库记录

### 方案
1. **前端防重**：点击后 disabled + loading 状态
2. **后端幂等**：入库单号生成后先去重检查
3. **事务保护**：整个入库操作为原子事务

---

## 7. 入库信息未进入库存列表

### 排查方向
1. **入库成功后 inventory 表是否更新**：
   - `inbound-v1.1.ts` 入库逻辑中是否同时 `UPDATE inventory SET stock = stock + ?`
2. **库存列表查询条件**：
   - `inventory-v1.1.ts` 查询是否限制 `status = 1`
3. **数据一致性**：
   - 入库记录 `inbound_records` 存在但 `inventory` 表无记录
